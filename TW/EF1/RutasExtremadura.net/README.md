# RutasExtremadura.net

## Descripción del Proyecto
RutasExtremadura.net es un portal web dedicado a proporcionar información y recursos relacionados con las rutas en Extremadura, España. El sitio incluye diversas secciones como Rutas, Novedades, Reservas, Histórico de Puntuaciones, Conecta4 y Bibliografía, ofreciendo a los usuarios una experiencia completa.

## Estructura del Proyecto
El proyecto está organizado en los siguientes directorios y archivos:

```
RutasExtremadura.net
├── css
│   └── style.css          # Contiene los estilos CSS del sitio web
├── js
│   ├── script.js          # Funciones generales de JavaScript para la navegación
│   └── conecta4.js        # Implementación del juego Conecta 4 en JavaScript
├── images
│   └── (imágenes del sitio)  # Directorio para las imágenes utilizadas en el sitio
├── pages
│   ├── rutas.html         # Contenido de la sección Rutas
│   ├── novedades.html     # Contenido de la sección Novedades
│   ├── reservas.html      # Formulario para realizar reservas
│   ├── historico.html     # Visualización de puntuaciones históricas
│   ├── conecta4.html      # Página del juego Conecta 4
│   └── bibliografia.html  # Referencias y enlaces a recursos
├── RutaExtremadura.html   # Página principal del sitio web
├── README.md              # Documentación del proyecto
└── TW_EF1_BibliografiaIA.pdf

## Funcionalidades
- **Menú de Navegación**: Un menú de navegación fácil de usar que enlaza a todas las secciones del sitio.
- **Diseño Responsivo**: El sitio está diseñado para ser accesible en diversos dispositivos.
- **Juego Conecta 4**: Un juego interactivo de Conecta 4 implementado en JavaScript.
- **Formulario de Reservas**: Un formulario para que los usuarios realicen reservas, con validación de entradas.
- **Contenido Dinámico**: Secciones como Histórico de Puntuaciones cargan datos de forma asíncrona.

## Instrucciones de Configuración
1. Clona el repositorio o descarga los archivos del proyecto.
2. Abre `RutaExtremadura.html` en un navegador web para visualizar el sitio.
3. Asegúrate de que todas las imágenes estén colocadas en el directorio `images` para que se muestren correctamente.

## Notas Adicionales
- El proyecto está desarrollado utilizando XHTML, CSS y JavaScript.
- Mejoras futuras pueden incluir rutas adicionales, características avanzadas en el juego y más contenido dinámico.
